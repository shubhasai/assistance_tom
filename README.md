# assistance_tom"Version-1.2"
It is a python based Virtual Assistance named Tom.
with features like
1. Information about anything from wikipedia
2. Opening Default Browser
3. Telling time
4. Play song on youtube
5. Telling Jokes
6. Along with Some funny Stuffs
